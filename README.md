基于 Spring Boot与Vue 的内容分享社区开发
本项目是一款基于前后端分离架构的生活方式分享社区 Web 应用。系统实现了从用户注册登录、图文与评论内容发布与管理，到首页瀑布流推荐及内容详情浏览的完整业务闭环，旨在提供高性能、响应式的用户内容交互体验。

【技术栈】

前端： Vue.js (配合 Vue Router, Vuex), Element UI, Axios

后端： Spring Boot 3.x, MyBatis, Spring Security, JWT

数据库： MySQL 5.7

【核心职责与技术实现】

前后端分离架构与接口设计： 负责系统整体架构搭建，基于 RESTful 规范定义并封装前后端 API 接口。在获取笔记列表与详情的数据交互中，采用“宽窄表”分离设计思想，列表页仅返回摘要及统计数据，详情页按需加载富文本与图片组，显著降低了网络传输开销与数据库连表查询压力。

无状态安全认证机制开发： 深度集成 Spring Security 与 JWT 技术。编写自定义 JwtAuthenticationFilter 拦截器，实现基于 Token 的无状态登录校验；全局配置 CorsConfig 解决跨域请求（CORS）问题，保障 API 接口调用的安全性与灵活性。

业务模块开发与防越权控制： 独立完成内容发布系统的后端服务与前端界面（集成富文本编辑器）。在笔记的更新与删除接口中，严格引入基于 Token 解析的当前登录用户 ID 校验机制，与数据库存储的 user_id 进行比对，从后端接口层面彻底杜绝了水平越权篡改他人内容的逻辑漏洞。

前端性能与交互体验优化： 运用 Vuex 进行全局状态管理（Token 与用户信息持久化）。在首页信息流模块，结合 CSS 布局与 JavaScript 滚动条事件监听，实现了图片的无限滚动懒加载与瀑布流展示，优化了前端渲染性能。
